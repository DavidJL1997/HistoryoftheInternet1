W3C and HTML and CSS

W3C

W3C is the World Wide Web Consortium. It is a international community where where they work together to develop Web standards for the World Wide Web. It is led by Tim Berners-Lee. The W3C is important because it upholds the World Wide Web and allows the public to surf the Internet with ease.

HTML

HyperText Markup Language(HTML) is a software language used by programmers. It is a set of markup symbols that are put into a file to be displayed on the internet. HTML is mostly used to create websites on the Internet. All the websites that are on the Internet contains HTML. With some presenting more HTML coding than others.

CSS

Cascading Style Sheets(CSS) is a coding langauge that correspondes to HTML. It describes how HTML elements are to be displayed on a website. CSS can be used to enhance the way websites look. Changing up the design and styles of websites to give it a unique look. CSS can be used with any XML- based markup language.
